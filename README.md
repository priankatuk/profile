# GitHub Pages Portfolio

This is a professional portfolio website designed for GitHub Pages. It features a clean, modern design with sections for:

- Personal profile and introduction
- Software project showcase
- Social media links
- Responsive design for all devices

## Setup Instructions

1. **Save the file**: Save the HTML file as `index.html` in your GitHub repository's root directory.

2. **Enable GitHub Pages**:
   - Go to your GitHub repository
   - Click on "Settings"
   - Scroll down to "Pages" section
   - Select "GitHub Pages" source: "Deploy from a branch"
   - Choose your main/master branch
   - Click "Save"

3. **Access your site**: After a few minutes, your site will be available at:
   ```
   https://<your-username>.github.io
   ```

## Customization

- **Personal Information**: Update the name, title, and social links in the header.
- **About Section**: Modify the profile content to reflect your background and skills.
- **Projects**: Add your own projects to the showcase grid, including project names, descriptions, technologies used, and links.
- **Profile Picture**: Replace the placeholder with your actual photo (update the `.profile-pic` div).

## Features

- Modern gradient design with shadow effects
- Fully responsive layout
- Project cards with technology tags
- Social media integration
- Clean typography and spacing

## Deployment

This site is optimized for GitHub Pages hosting. Simply push the `index.html` file to your repository and enable GitHub Pages in the repository settings.

## Browser Compatibility

The site works on all modern browsers including Chrome, Firefox, Safari, and Edge.
